# TBProgressBar仿淘宝进度条(无文字)


![demo](demo.gif)

# 主要方法:
## 1.设置进度 范围0 - 100
    setProgress(int progress)
## 2.开始 关闭 演示动画
    ```javastartShowAnimate()  stopShowAnimate()```      
## 3.设置进度背景颜色
    ```java setBgColors(int color1, int color2)```
## 4.设置置进度条前景颜色
    ```java setForegroundColors(int color1, int color2)```
